# Evolution_Tree
一个可视化，交互式，动态制作进化树的软件

# v1.0.1

iTOL (Interactive Tree Of Life)需要用户Newick 或 Nexus 格式的文本文件才能做出进化树，而我们想要用户可以自己diy进化树，根据自己的设想动态、交互式地“画”一棵树。
但提供和itol一样专业的功能：
## （1）布局与形态控制 (Layouts)
### 矩形树 (Rectangular Tree)
### 环形/圆形树 (Circular Tree)
### 无根/放射树 (Unrooted/Radial Tree)
## （2） 演化分支的“色彩标签化” (Branch & Leaf Coloring)
要点：不只是单条线改颜色，而是支持“一键为整个单系群（Monophyletic group）渲染背景色”。允许用户为某个节点的所有后代分支覆盖一层半透明的扇形（环形树）或矩形（矩形树）背景色块，用来区分不同的“门（Phylum）”或“属（Genus）”（例如将所有大阳化石层位物种标为淡蓝色）。
## （3）分支折叠与几何简化 (Branch Collapsing)
要点：当某个分类下物种过多、但不是研究重点时，将其收起。
表现： 会将折叠后的复杂分支渲染为一个等腰三角形（Triangle）。三角形的横向长度代表该分支内演化最快的物种的时间跨度；三角形的纵向底边宽度代表该分支内包含的物种数量。这种几何简化极具科学严谨性。
## （4）文本对齐与引导线 (Label Alignment & Dashed Lines)：
当开启“物种对齐最右侧”时，物种名称右对齐，而真实的树枝末端到名字之间，必须用极细的灰色虚线（Dashed Alignment Lines）相连。这样既能看出真实的灭绝时间，又能保证文字排版整齐。
## （5）用户可以粘贴名单——按行分割形成业主节点/子节点
## （6）用户可以拖拽主节点/子节点连接到父节点上
## （7）可以隐藏中间的非叶子节点的名字

# v1.0.3
## 父节点/内部节点名称现在会画在自己的树枝上方，不再像叶子节点一样贴在枝端。
## 支持空白无名字节点：可以点“添加空白无名字节点”，也可以把节点名清空；空白节点会显示一个小圆点手柄，仍然能选中、拖拽、改父节点。
## 新增 Ctrl+Z 撤销。
## 新增 Ctrl+C 多选复制子树
## 文本颜色和分支线颜色已分开：文本默认黑色
## 多选增强：
空白处直接拖矩形可框选多个分支/节点；按住 Shift 或 Ctrl 点节点、圆点、分支线都可追加多选。分支线还加了隐形加粗命中区域，更容易点中。
## 拖放父节点更敏感
释放时会在鼠标周围一定半径内吸附最近的可见父节点，不必精准压中文字上。父节点接收子节点后，文字不再大幅跳到分支中段，只会轻微保持在树枝上方
